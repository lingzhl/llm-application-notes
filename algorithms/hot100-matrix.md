## 核心思路

利用矩阵的两个性质：

- 每一行递增
- 每一列递增

从 **右上角开始搜索**：

- 当前值 == target → 返回 True
- 当前值 > target → 左移（变小）
- 当前值 < target → 下移（变大）

直到越界结束。

## 为什么这么做

右上角具有特殊性质：

- 左边更小
- 下边更大

所以比较一次后，**移动方向唯一确定**，避免回头搜索。

## 模板代码

```python
def search_matrix(matrix, target):
    if not matrix:
        return False

    m, n = len(matrix), len(matrix[0])
    i, j = 0, n - 1

    while i < m and j >= 0:
        if matrix[i][j] == target:
            return True
        elif matrix[i][j] > target:
            j -= 1
        else:
            i += 1

    return False
````

## 知识点总结

### 1. 二维有序矩阵

* 行递增
* 列递增

---

### 2. Z 字型查找

每次排除一行或一列

---

### 3. 为什么不用二分？

二维不是一维连续结构，直接二分不成立（除非完全展开）

---

### 4. 起点选择

✔ 右上角 / 左下角
❌ 左上角 / 右下角

---

## 面试官高频问题

### 1. 为什么从右上角？

因为左小下大 → 方向唯一

---

### 2. 时间复杂度？

👉 O(m + n)

因为最多：

* 向左 n 次
* 向下 m 次

---

### 3. 空间复杂度？

👉 O(1)

只用指针变量

---

### 4. 为什么不从左上角？

因为两个方向都变大，无法判断走哪边

---

### 5. 左下角可以吗？

可以：

* 大了 → 上
* 小了 → 右

---

## 复杂度

* 时间复杂度：`O(m + n)`
* 空间复杂度：`O(1)`

---

## 面试标准回答

这题利用矩阵行列递增的性质，从右上角开始搜索。如果当前元素等于目标直接返回；如果大于目标就左移，因为这一列下面只会更大；如果小于目标就下移，因为这一行左边只会更小。每一步都能排除一整行或一整列，因此时间复杂度是 O(m + n)，空间复杂度是 O(1)。
